# Kızıl Kod — Çin Teknoloji Stratejisi

Bu depo, "Kızıl Kod" projesinin kaynak ve dokümantasyon iskeletini içerir. Amaç: Türkiye ve küresel bağlamda Çin'in teknoloji stratejilerini inceleyen, analiz eden ve politika/teknoloji önerileri üreten bir çalışma seti geliştirmek.

Özellikler
- Proje iskeleti: TypeScript tabanlı küçük bir araç ve test iskeleti.
- Dokümantasyon ve katkı rehberi için başlangıç dosyası.

Hızlı başlangıç (Node.js gerektirir)

1. Bağımlılıkları yükle

```bash
npm install
```

2. Geliştirme sunucusu

```bash
npm run dev
```

3. Derle ve çalıştır

```bash
npm run build
npm start
```

4. Testleri çalıştır

```bash
npm test
```
